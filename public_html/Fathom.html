Fathom Analytics

Using Fathom on baty.net for web analytics. This is hopefully able to replace Matomo, which is a bit heavy. 1.0 was released this week.

https://github.com/usefathom/fathom

Set up according to configuration instructions in Github.

It's in /home/jbaty/apps/fathom

Configured to launch at startup using: 

[Unit]
Description=Starts the fathom server
Requires=network.target
After=network.target

[Service]
Type=simple
User=jbaty
Restart=always
WorkingDirectory=/home/jbaty/apps/fathom
ExecStart=/usr/local/bin/fathom server

[Install]
WantedBy=multi-user.target
