Added: <2015-09-29 Tue>

Workflow

When processing a roll of film, don't forget to do the following:

  • Tag and caption using Photo Mechanic
  • Rename photos with (Film Roll Sequence) YYYY Roll-00NHEADLINE using Photo Mechanic
  • Use exiftool to add camera info See Adding Exif data to scans
  • Use exiftool to fix Date Taken exiftool -v "-FileModifyDate>DateTimeOriginal" -overwriteoriginal .
  • Move to Auto Import folder for Import into Lightroom
  • Move to dated folder named like YYYY Roll-00N
  • Upload to Flickr
  • Add to logbook

Scanning

  • Using The Pakon

    Bought my (good one) from SerranoRay Position the DX code on top and lowest frame number first. Sometimes the scanner isn't found properly. So far, the following seems to work best:

    • Boot Windows XP
    • Turn on scanner
    • Start PSI

    For Black and White, the film goes in numbers up and first-frame-first

Digital Processing Steps

  • For iPhone, plug into iMac and let AutoImporter do the work (Inbox)
  • For camera, ingest with Photo Mechanic to _Inbox
  • Caption with Photo Mechanic and apply Stationery Pad
  • Rename in Photo Mechanic (YYYY-MM-DDheadline e.g. {iptcyear4}-{iptcmonth0}-{iptcday0}headline:lc)
  • Move to Lightroom's Auto Import folder
  • Move in Lightroom to YYYY/MM-Month YYYY/ folder

Sync and Backup

Sync

I do not use iCloud sync. Instead, I import from the phone and delete. Once processed on the Mac and moved into the dated folders, Google Photos syncs them. I then use Google Photos on devices and other Macs to view and find photos.
